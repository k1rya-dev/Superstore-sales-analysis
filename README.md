# Анализ данных о продажах компании Superstore

## Описание проекта

В рамках работы был проведен анализ данных о продажах и прибыли компании Superstore. 
Целью являлось выявление ключевых бизнес-показателей, оценка эффективности различных 
категорий товаров и регионов, а также исследование влияния скидок на прибыльность продаж.

Анализ выполнен с использованием языка Python в среде Jupyter Notebook. 
Для обработки и визуализации данных применялись библиотеки pandas, matplotlib и seaborn.

## Структура проекта

Репозиторий содержит следующие файлы:

superstore_analysis.ipynb - основной файл с кодом и визуализациями
superstore_sales.csv - исходный набор данных

## Запуск

Для воспроизведения анализа необходимо выполнить следующие шаги.

Клонировать репозиторий:

git clone https://github.com/ваш-username/superstore-analysis.git
cd superstore-analysis

Установить необходимые библиотеки:

pip install pandas matplotlib seaborn

Запустить Jupyter Notebook:

jupyter notebook

Открыть файл superstore_analysis.ipynb и выполнить все ячейки последовательно.

## Результаты

В ходе анализа были получены следующие выводы.

По категориям товаров:
- Наиболее прибыльная категория - Technology с показателем 145454.95 у.е.
- Категория Office Supplies принесла 122490.80 у.е.
- Наименьшая прибыль у категории Furniture - 18451.27 у.е.

По регионам:
- Лидирующий регион по продажам - West с объемом 725457.82 у.е.
- Наименьшие продажи зафиксированы в регионе South - 391721.90 у.е.
- Три штата (Техас, Нью-Йорк, Калифорния) составляют 40.9% от общих продаж.
- Три наименее результативных штата (Северная Дакота, Западная Вирджиния, Мэн) дают только 0.15% от общего объема.

По убыточным позициям:
- Всего обнаружено 1871 убыточная позиция.
- Наибольший убыток принес товар Riverside Palais Royal Lawyers Bookcase.

По скидкам:
- Скидки не стимулируют рост продаж (корреляция -0.03).
- Скидки слабо коррелируют с убытками (корреляция -0.22).
- При скидках свыше 40% все 933 транзакции оказались убыточными.

По сезонности:
- Пик продаж приходится на ноябрь (до 118448 у.е. в 2017 году).
- Корреляция между сезонами разных лет составляет -0.14, что указывает на низкую предсказуемость.

По дням недели:
- День недели существенно не влияет на прибыль.
- Медианная прибыль по дням варьируется в пределах 8-10 у.е.

## Рекомендации

На основе проведенного анализа предложены следующие меры.

1. Установить максимальный порог скидки на уровне 20-25% для предотвращения убытков.
2. Сосредоточить усилия на развитии продаж в регионах с низкой долей рынка.
3. Увеличить присутствие в категории Technology как наиболее прибыльной.
4. Пересмотреть ценообразование в категории Furniture для повышения рентабельности.
